# Git Branch MenuBar

メニューバーに現在のGitブランチを表示するシンプルなmacOSステータスメニューアプリです。\
1つのリポジトリを選択して監視します。

## 主な機能

- メニューバーに現在のブランチ名を表示
- 表示対象リポジトリをフォルダ選択で設定
- 5秒ごとの自動更新（起動時に開始）
- 手動更新（\`更新\`）
- Gitリポジトリでない場合は選択不可
- detached HEAD 時は `DETACHED(\`hash\`)` 表示
- 起動時自動実行（\`ログイン時に起動\`）トグル

## 動作要件

- macOS 12 以上で実行可能
- Xcode + Swift 5.9 以上推奨
- 対応対象は1リポジトリのみ

## 使い方

1. アプリを起動すると、メニューバーに `No repo selected` と表示されます。
2. メニューを開いて `リポジトリを選択...` から対象リポジトリを選びます。
3. 選択中リポジトリのブランチが `repo:branch` 形式でメニューバーに表示されます。
4. 必要に応じて `更新` を押して即時再取得します。
5. `選択を解除` で監視対象を外します。
6. `ログイン時に起動` でログイン時の自動起動をON/OFFします（macOS 13+）。

## メニュー構成

- `リポジトリを選択...` : 監視対象を選択
- `選択を解除` : 監視対象を外す
- `ログイン時に起動` : 自動起動のON/OFF（macOS 13+）
- `更新` : 手動でブランチ再取得
- `リポジトリを開く` : Finderで対象フォルダを開く
- `終了` : アプリ終了

## 補足

- 選択したリポジトリのパスは UserDefaults に保存され、次回起動時に復元されます。
- Gitコマンド実行にはシステムの `git` を使用します。
