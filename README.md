# Git Branch MenuBar

メニューバーに現在のGitブランチを表示するシンプルなmacOS常駐アプリです。
1つのリポジトリを選択して監視します。

## 主な機能

- メニューバーに現在のブランチを表示
- 表示対象リポジトリをフォルダ選択で設定
- 5秒ごとの自動更新（起動時に開始）
- 手動更新（`🔄 更新`）
- `🗑️ 選択を解除` で監視対象を外す
- detached HEAD 時は `DETACHED(<hash>)` 表示
- `🚀 ログイン時に起動` でログイン時自動起動をON/OFF（macOS 13+）

## 動作要件

- macOS 12 以上
- Swift 5.9 以上推奨
- 対応対象は1リポジトリのみ

## 使い方

1. `swift run` で起動すると、メニューバーに `⚪ No repo selected` と表示されます。
2. メニューで `📁 リポジトリを選択...` を選び、監視したいGitリポジトリを指定します。
3. 選択したリポジトリのブランチが `repo:branch` 形式で表示されます。
4. `🔄 更新` で即時再取得します。
5. `🗑️ 選択を解除` で監視対象を外します。
6. `🚀 ログイン時に起動` でログイン起動をON/OFFできます（macOS 13+）。

## メニュー構成

- `📁 リポジトリを選択...`
- `🗑️ 選択を解除`
- `🚀 ログイン時に起動`
- `🔄 更新`
- `📂 リポジトリを開く`
- `🚪 終了`

## ビルド・実行

```bash
cd /Users/ayumu.hisada/Github/mac_branch
swift build
swift run
```

## 補足

- 選択したリポジトリパスは `UserDefaults` に保存され、次回起動時に復元されます。
- Gitコマンドにはシステムの `/usr/bin/git` を使用します。
